display:none

visibility: hidden

1- 元素在页面中仍占据空间 2- 不会触发绑定事件

opacity: 0

1- 元素在页面中仍占据空间 2- 能够触发绑定事件

position: absolute

1- 通过将元素移出可视区来实现

z-index: -1

1- 通过其他元素盖住该元素

clip-path: circle(0);

1- 使用元素裁剪的方法来实现元素隐藏 2- 元素仍然在页面中占据位置 3- 会触发绑定事件

transform: scale(0)

1- 将元素缩放为0 2- 元素仍然在页面中占据位置 3- 会触发绑定事件
